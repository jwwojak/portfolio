## Flex robot

The [Opentrons Flex Instruction Manual](https://docs.opentrons.com/flex/) is the primary guide for Opentrons' flagship liquid-handling robot. This guide is an ongoing collaborative effort with contributions from the entire writing team. Specific examples as an individual contributor are linked below.

- [Unboxing](https://docs.opentrons.com/flex/installation/unboxing/) explains how to set up a new Flex robot.

- [Flex Log Files](https://docs.opentrons.com/flex/advanced-operation/log-files/) explains the purpose of different log files generated by the robot and how to download them.

- [Command Line Operation Over SSH](https://docs.opentrons.com/flex/advanced-operation/command-line/) explains how to make a secure SSH connection to a Flex robot.

## OT-2 robot

The [Opentrons OT-2 Instruction Manual](https://docs.opentrons.com/ot-2/) is the guide for Opentrons' older, entry-level liquid-handling robot. I am the primary author for the recent version of this manual.

## Robot modules

Modules are hardware components that extend the capabilities of the Flex and OT-2 robots. Selected works as the primary writer include:

- [Flex Stacker Instruction Manual](https://docs.opentrons.com/stacker/)
- [Absorbance Plate Reader Instruction Manual](https://docs.opentrons.com/absorbance-plate-reader/)
- [HEPA/UV Module Instruction Manual](https://docs.opentrons.com/hepa-uv/)
